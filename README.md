SafeCity :-
  A Real-Time City Emergency Response System built using the MERN Stack.

SafeCity is a web application that helps citizens quickly report emergencies and allows authorities to respond efficiently.
The platform connects citizens with nearby Police Stations, Fire Stations, and Hospitals through a centralized dashboard.

Features :-
  🚨 Report Emergencies
  📍 Live Location Detection
  🗺️ Interactive Maps
  🚓 Police Dashboard
  🚒 Fire Department Dashboard
  🏥 Hospital Dashboard
  👨‍💼 Admin Panel
  📊 Emergency Analytics
  🔔 Real-Time Status Updates
  🔒 Secure Authentication

Tech Stack :-
  Frontend : React, Vite, Tailwind CSS
  Backend : Node.js, Express.js
  Database : MongoDB
  Other : JWT Authentication, Leaflet

Workflow
  Citizen reports an emergency.
  System captures the user's location.
  Emergency is stored in MongoDB.
  Nearby department receives the alert.
  Department updates the response status.
  Citizen tracks the emergency status.
